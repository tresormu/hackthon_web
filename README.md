# MamaCare+ Doctor's Dashboard

A comprehensive maternal and child health companion system designed for doctors to track and manage pregnant patients and infants.

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```

## Core Features
- **Overview Dashboard**: Analytics on patient registration and care status.
- **Patient Management**: Enroll new mothers and track their pregnancy journey.
- **Escalation Module**: Automated alerts for missed appointments.
- **CHW Coordination**: Interface to coordinate with community health specialists.
- **Postnatal Care**: Vaccination and growth tracking for infants.

## Tech Stack
- React + Vite + TypeScript
- Tailwind CSS (Styling)
- Framer Motion (Animations)
- Lucide React (Icons)
- Recharts (Data Visualization)
